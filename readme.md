#乐乐音乐4.0简洁版
开发环境：android studio
#支持音频格式
mp3、ape、flac、wav等
#支持歌词格式


- ksc歌词：卡拉OK歌词


- hrc歌词：happy lyrics歌词，乐乐音乐4.0自定义的动感歌词格式，可准确到歌词每个字。
#完成功能

- 音频播放、定位播放

- 动感歌词显示

- 流畅、平滑的歌词滚动
- 支持歌词字体大小、颜色修改、歌词换行
- 支持歌词进度修改，可快进+0.5秒，-0.5秒

#Hrc歌词制作器（播放器）
https://github.com/zhangliangming/HappyPlayer-PC.git
